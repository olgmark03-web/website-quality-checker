# 🔍 Website Quality & GDPR Compliance Checker

Ένα αυτοματοποιημένο εργαλείο ανάλυσης ιστοσελίδων, το οποίο ελέγχει την ποιότητα, την προσβασιμότητα και τη συμμόρφωση με τον Γενικό Κανονισμό Προστασίας Δεδομένων (GDPR). Αναπτύχθηκε ως **πτυχιακή εργασία**, συνδυάζοντας τεχνικές web scraping, ανάλυσης περιεχομένου και αυτοματοποιημένης δοκιμής.

---

## Βασικές Λειτουργίες

*   **Αυτοματοποιημένος Έλεγχος:** Αναλύει μια λίστα ιστοσελίδων (από αρχείο `websites_to_analyze.txt`) και παράγει μια αναλυτική αναφορά.
*   **Έλεγχοι Ποιότητας:**
    *   **Απόδοση:** Έλεγχος ταχύτητας φόρτωσης και μεγέθους πόρων.
    *   **Προσβασιμότητα:** Ανίχνευση βασικών προβλημάτων (π.χ. εναλλακτικό κείμενο σε εικόνες).
    *   **Δομή & SEO:** Ανάλυση επικεφαλίδων, meta tags και βασικών στοιχείων δομής.
*   **Έλεγχοι Συμμόρφωσης (GDPR):**
    *   Εντοπισμός πολιτικής απορρήτου (privacy policy).
    *   Έλεγχος για banners αποδοχής cookies.
    *   Ανίχνευση φορμών συλλογής προσωπικών δεδομένων.
*   **Αναφορά Αποτελεσμάτων:** Δημιουργία λεπτομερούς αρχείου καταγραφής (`website_checker_history_test.txt`) με τα ευρήματα για κάθε ιστοσελίδα.

---

## Τεχνολογίες & Εργαλεία

*   **Γλώσσα:** Java (JDK 11+)
*   **Framework:** JavaFX (για ενδεχόμενη μελλοντική GUI)
*   **Βιβλιοθήκες:**
    *   **Selenium WebDriver:** Για αυτοματοποιημένη πλοήγηση και αλληλεπίδραση με την ιστοσελίδα.
*   **Εργαλεία:** Maven (για build και διαχείριση dependencies).

---


### Προαπαιτούμενα

*   **Java Development Kit (JDK)** έκδοση 11 ή νεότερη.
*   **Maven** εγκατεστημένο (ή χρήση του wrapper).
*   **Google Chrome** browser εγκατεστημένο.
*   **ChromeDriver** (συμβατό με την έκδοση του Chrome σας).
